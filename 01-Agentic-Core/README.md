# 01 - Agentic Core

## Overview
The Agentic Core serves as the engine of the Kyndryl Agentic AI Framework, responsible for orchestrating, securing, and scaling enterprise technology into agentic AI workflows.

## Key Topics
- **Orchestration**: Coordinating AI agents across enterprise systems
- **Workflow Design**: Creating efficient agentic workflows
- **Governance**: Ensuring compliance and control
- **Scaling Strategies**: Growing from pilot to production

## Learning Objectives
- Understand the core principles of agentic AI orchestration
- Learn how to design secure and transparent workflows
- Master governance frameworks for AI agents
- Develop strategies for scaling agentic systems

## Structure
- `notes/` - Detailed learning notes and concepts
- `examples/` - Practical orchestration examples
- `references/` - Links to documentation and resources
